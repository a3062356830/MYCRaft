# CareerCraft

AI 驱动的职业模拟沙盒，像素风职场模拟器。

## 项目定位

CareerCraft 不是题库，而是一个 AI 驱动的职业模拟沙盒。用户进入职业世界，选择职业岛屿，接受 AI 发布的职业任务，查看学习资源，提交作品，获得 AI 评审，点亮技能树并解锁新任务。

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- App Router
- Zustand (后续)

## 页面规划

1. **职业选择大厅** `/lobby`
   - 展示 CareerCraft 世界地图
   - 展示两个职业入口：软件工程岛、数据山脉
   - 点击进入对应职业大厅

2. **职业中枢页面** `/career/[careerId]`
   - 左侧：技能树
   - 中间：任务看板
   - 右侧：资源书架 + AI导师

3. **任务详情**
   - 展示任务背景、目标、交付物、评分标准
   - 用户可以接受任务

4. **资源推荐**
   - 用户点击"寻找资源"
   - 展示 RAG 推荐学习资料

5. **任务提交**
   - 用户提交文本报告或代码内容
   - 第一版不做真实文件上传

6. **AI反馈页面**
   - 展示评分、评语、技能经验、头衔徽章
   - 技能树节点点亮

7. **个人档案** `/profile`
   - 展示等级、经验、已获得头衔
